CÓDIGO EXPLICADO:

No código, há 6 partes:

index.php: Nesta parte é a tela inicial, onde é feito o cadastro de usuário

atribuir_tarefa.php: aqui é onde depois de cadastrar o usuário, nós iremos atribuir as tarefas, como: setor, descrição e prioridade

gerenciar_tarefas.php: dps de adicionar a atrefa, vamos para a tela onde irá aparecer o gerenciamento das tarefas, como se fosse a tela do kanban, podemos editar e excluir a tarefa

style.css: onde está localizado o style do gerenciamento

excluir_tarefa.php: código para consegurimos excluir a tarefa

editar_tarefa.php: código para conseguir editar a tarefa, e mudar no banco de dados
